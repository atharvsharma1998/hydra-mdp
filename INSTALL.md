# Install

Use **[`QUICKSTART.md`](QUICKSTART.md)** — it covers environment setup and the full
`pth → Python → ONNX → TensorRT → C++` path on `main`.

```bash
git clone https://github.com/atharvsharma1998/hydra-mdp.git
cd hydra-mdp
# stay on main
```

For training and full NAVSIM PDM evaluation (needs dataset download), see
[`docs/TRAINING.md`](docs/TRAINING.md).

Upstream NAVSIM map/data download notes (only if you train or run full eval):
[`docs/install.md`](docs/install.md).
